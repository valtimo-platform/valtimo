## Features

## Enhancements

* **Access control overview tab**

  A new overview tab presents the permissions in a human-readable manner.

* **Metroline widget available for IKO widget tabs**

  The metroline widget can now be added to IKO widget tabs, allowing case progress to be visualised alongside other
  case information shown to customers.

## Documentation

* **Development, security, and API guidelines published**

  Four new documents have been added as part of NCSC audit rule U/WA.01-1 (development,
  maintenance, decommissioning, security and data-processing guidelines):
  [
  `backend/CODING-GUIDELINES.md`](https://github.com/valtimo-platform/valtimo-platform/blob/main/backend/CODING-GUIDELINES.md),
  [API design standards](../../../contributing-to-valtimo/api-design-standards.md),
  [Deprecation policy](../../../contributing-to-valtimo/deprecation-policy.md),
  and [Data processing guidelines](../../../contributing-to-valtimo/data-processing-guidelines.md).
  The release-notes template now includes `## Deprecations` and `## Removals` sections.
