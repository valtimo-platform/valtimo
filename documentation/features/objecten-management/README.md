# 📋 Objecten management

Object Management allows you to configure connections to external Objecten API instances and display their objects
within Valtimo.

## Features

* **Object lists** — Browse and search objects from configured Objecten APIs
* **Object forms** — Display object details using Form.io forms
* **Form.io component** — Select objects from within forms using the Object Management Select component
* **Access control** — Permission-based access to configurations (optional, behind feature flag)
