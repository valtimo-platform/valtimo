# Evenementenvergunning demo (frontend)

The web interface for the event-permit (*evenementenvergunning*) demo published as
`ritense/gzac-evenementenvergunning-frontend`.

## How to start

From the `frontend` folder (after `npm install`, and with the matching backend
running):

```bash
npm run start:evenementenvergunning
```

Then open <http://localhost:4200>.
