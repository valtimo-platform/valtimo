# Evenementenvergunning demo (backend)

A demo application that shows how to build a real case type on Valtimo — here,
handling event-permit (*evenementenvergunning*) requests. It is the in-repo
version of the standalone demo:
<https://github.com/generiekzaakafhandelcomponent/gzac-evenementenvergunning-demo-backend>.

Released as `ritense/gzac-evenementenvergunning-backend`.

## How to start

With the supporting services already running, from the repository root:

```bash
./gradlew :backend:app:evenementenvergunning:bootRun
```
