# Evenementenvergunning demo (backend)

A demo application that shows how to build a real case type on Valtimo — here,
handling event-permit (*evenementenvergunning*) requests.

Published as `ritense/gzac-evenementenvergunning-backend`.

## How to start

From the repository root:

```bash
./gradlew :backend:apps:evenementenvergunning:bootRunWithDocker
```

This starts the supporting services in Docker and then launches the
application.
